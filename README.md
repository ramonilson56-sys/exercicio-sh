## Como Baixar e Instalar

- Clique em "Code" e depois em "Download ZIP" para baixar os arquivos.
- Extraia os arquivos em uma pasta no seu computador.
- Navegue até essa pasta no terminal.
- Siga as instruções abaixo para executar o programa.


## Como Executar a Calculadora

1. Certifique-se de ter o Python instalado.
2. Faça o download dos arquivos do repositório.
3. No terminal, navegue até a pasta do projeto e execute:

python3 calculadora.py
